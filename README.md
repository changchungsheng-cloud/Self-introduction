## 🧠 Research Overview
I am currently working on a vision-based poultry health monitoring system using a multi-camera mobile robot.  
The system actively interacts with the environment and analyzes behavioral responses to estimate health conditions.  

My research focuses on:
- Multi-camera perception and BEV (Bird’s Eye View) transformation  
- Behavior-based analysis using vision data  
- Integration of perception and action in embedded robotic systems
